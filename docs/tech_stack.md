# Technology Stack

## Core

- **Build Tooling**: [Vite](https://vite.dev/)
- **Framework**: React
- **Runtime**: Node.js (v20.19+ or v22.12+ recommended)
- **Language**: TypeScript

## User Interface

- **Component Library**: [shadcn/ui](https://ui.shadcn.com/)
- **Styling Engine**: [Tailwind CSS](https://tailwindcss.com/)
- **Typography**: [Excalifont](https://github.com/excalidraw/excalidraw/tree/master/public/fonts) (Self-hosted woff2)

## Visualization

- **Rendering**: HTML5 Canvas (2D Context) or SVG (via React state).
- **Animation**: CSS transitions for UI; Frame-based or state-driven rendering for algorithms.

## Infrastructure

- **Package Manager**: `npm` (or `bun` if preferred locally, but `npm` is standard).
- **Deployment**: Static hosting (GitHub Pages, Vercel, Netlify).
